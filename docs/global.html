nextpow2(num)

Source:
  • lib/math/nextpow2.js, line 11

Returns the ceil of the log2 of the absolute value of the passed number

Example
nextpow2(8); // 3
nextpow2(9); // 4
nextpow2(16); // 4
nextpow2(30); // 5
nextpow2(0); // -Infinity
