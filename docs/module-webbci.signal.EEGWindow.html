webbci.signal. EEGWindow

new EEGWindow(size, numChannels, callback)

Create a new EEGWindow object.
Parameters:
Name Type Description
size number The number of samples to be stored before callback is called.
numChannels number The number of channels in each sample.
callback requestCallback Called when the EEGWindow has a number of samples equal to size. An array of dimensions channels x samples is passed to the callback function.
